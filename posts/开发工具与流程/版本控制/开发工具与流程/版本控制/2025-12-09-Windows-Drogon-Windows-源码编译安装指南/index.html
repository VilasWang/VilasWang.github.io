8. 🔧 常见问题

8.1 编译错误

  • VS 版本不匹配: 确认 compiler.version 与安装的 Visual Studio 版本一致
  • C++ 标准问题: 确保 compiler.cppstd 设置正确
  • 依赖缺失: 使用 --build=missing 强制构建缺失的依赖

8.2 链接错误

  • DLL 找不到: 检查 PATH 环境变量设置
  • 库文件路径: 确认 CMAKE_PREFIX_PATH 设置正确

8.3 运行时错误

  • 配置文件: 检查 config.json 配置是否正确
  • 端口占用: 确认配置的端口未被占用
